# Package Feed Search

Search the live Avocado Linux package feed at <code>repo.avocadolinux.org</code> directly from
your browser. This is the same metadata that <code>avocado sdk dnf search</code> queries inside
the SDK container — no installation required.

Pick a target, type a package name (or part of a summary), and copy the install command for
the match.
