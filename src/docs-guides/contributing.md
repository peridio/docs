# Contributing References

We welcome community contributions to the [Avocado OS references repository](https://github.com/avocado-linux/references).

## How to Contribute

1. Fork the [references](https://github.com/avocado-linux/references) repository
2. Create a new directory for your reference project
3. Include an `avocado.yaml`, application source, and build scripts
4. Add a `README.md` with frontmatter describing your reference
5. Add a `getting_started.md` with setup instructions
6. Submit a pull request

More detailed contribution guidelines coming soon.
