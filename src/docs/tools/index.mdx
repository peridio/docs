# Tools

Peridio provides a comprehensive suite of tools to accelerate embedded Linux development and simplify IoT fleet management. From building your first system image to managing millions of devices in production, these tools integrate seamlessly to support your entire product lifecycle.

## Avocado Linux

Fast-track your embedded Linux development with production-ready build systems and streamlined workflows. Avocado tools eliminate months of Yocto bring-up while maintaining full flexibility for customization.
