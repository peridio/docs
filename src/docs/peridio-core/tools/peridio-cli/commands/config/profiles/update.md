```
Usage: peridio config profiles update [OPTIONS] <PROFILE_NAME>

Arguments:
  <PROFILE_NAME>  The profile name to update

Options:
      --api-key <API_KEY>          Update the API key. Can be omitted to supply the value interactively
      --base-url <BASE_URL>        Update the base URL
      --ca-path <CA_PATH>          Update the CA path
      --api-version <API_VERSION>  Update the API version
      --no-input                   Disable interactivity
  -h, --help                       Print help

```
