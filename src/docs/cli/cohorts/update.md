# peridio cohorts update

Update an cohort.

## Flags

`-h`, `--help`

Prints help information.

### Options

`--description <description>`

The cohort description.

`--name <name>`

The name of the cohort.

### Required

`--cohort_prn <cohort_prn>`

The Peridio Resource Name (PRN) of the cohort.
