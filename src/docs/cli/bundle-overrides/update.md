Usage: peridio bundle-overrides update [OPTIONS] --prn <PRN>

Options:
      --prn <PRN>                  The PRN of the bundle override to update
      --name <NAME>                The name of the bundle override
      --description <DESCRIPTION>  The description of the bundle override
      --ends-at <ENDS_AT>          The end date/time for the bundle override (ISO 8601 format)
      --starts-at <STARTS_AT>      The start date/time for the bundle override (ISO 8601 format)
      --bundle-prn <BUNDLE_PRN>    The PRN of the bundle to override
  -h, --help                       Print help
