# peridio signing-keys create

Create a signing key.

## Flags

`-h`, `--help`

Prints help information.

## Options

### Required

`--key <key>`

The base64 encoding of the raw 32 byte public key.

`--name <name>`

The name of the key.

`--organization-name <organization-name>`

The organization to interact with.
