## Avocado CLI 0.16.0

### New features

- Add support for setting multiple `stone-input-paths`.
- Enable overriding `stone_manifest` in runtime config.
- Update `provision` to support `--out` flag for output control.

### Improvements

- Identify exact prebuilt extension names and version.
