## Avocado CLI 0.14.0

### New features

- Add `--reference` flag to `avocado init` to start a project from a GitHub reference.
- Update `avocado ext checkout` to support versioned extensions.

### Improvements

- `avocado init` now generates a `.gitignore` file.
