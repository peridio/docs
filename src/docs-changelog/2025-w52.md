## Avocado CLI 0.20.0

### New features

- Implement `lock.json` file for tracking DNF dependencies.

### Improvements

- Use JCS (JSON Canonicalization Scheme) for lockfile storage.
- Move SDK bootstrap, rootfs install, and target-sysroot install to `sdk install`.
- Remove remaining bootstrap logic from container entrypoint.
- Preserve file attributes on extension overlay merging.
- Add warning output for config parsing errors.

## Avocado CLI 0.19.1

### Bug fixes

- Use SDK `container_args` for provision commands.
- Fix Windows builds.

### 0.19.0

### New features

- Add stamps for tracking task dependencies.
- Add support for extension `on_unmerge` commands.
- Add `AVOCADO_ENABLE_SERVICES` to extension metadata.

### Improvements

- Update `enable-services` to call `nativesdk-systemctl`.
- Add stamps and deps for extension image creation.
- Update build platform scripts to explicitly pass `AVOCADO_EXT_LIST`.
- Move provision state to `.avocado` directory.
