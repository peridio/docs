## Improvements

- Support nested configs in `avocado fetch`.
- Preserve `volume_info` during builds.

## Bug fixes

- Remove extraneous RPM database location overrides.

## 0.10.1

## Improvements

- Update `fetch` to pull latest Docker container.
- Prevent extension packages from having dependencies.
- Prevent extension packages from acquiring automatic external dependencies.
- Remove rootfs RPM database copy step.

## 0.10.0

## Improvements

- Use the host architecture to choose the default QEMU architecture.
- Minimize SDK bootstrapping packages.
- Run scriptlets during SDK bootstrap.
- Add `--no-bootstrap` flag to `sdk run` to skip container SDK install.

## Bug fixes

- Fix target-specific runtime dependency inclusion.
