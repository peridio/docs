# NXP FRDM-i.MX 95

Freedom development platform for the i.MX 95 applications processor with eIQ Neutron NPU

**6x Cortex-A55 @ 2.0GHz + Cortex-M7 + Cortex-M33** | **2.0 TOPS NPU** | **8GB LPDDR4X** | **32GB eMMC**

<a href="https://www.nxp.com/design/design-center/development-boards-and-designs/i-mx-evaluation-and-development-boards/freedom-development-platform-for-i-mx-95:FRDM-IMX95" target="_blank" rel="noopener noreferrer">View on NXP.com</a>

## Overview

The FRDM-i.MX 95 is NXP's Freedom development board for the i.MX 95 applications processor family, integrating six Cortex-A55 cores, Cortex-M7 and M33 safety-domain cores, an Arm Mali GPU, a 2.0 TOPS eIQ Neutron NPU, 4K VPU, and ISP. It provides a comprehensive evaluation platform with Gigabit Ethernet, USB 3, PCIe Gen3, and an M.2 slot for Ara-series AI accelerator expansion.
