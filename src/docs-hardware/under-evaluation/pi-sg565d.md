# Quectel Pi H1 (PI-SG565D)

Developer-friendly single-board computer with Qualcomm QCS6490 for edge AI and IoT prototyping

**Octa-core Kryo 670 @ 2.7GHz** | **12 TOPS** | **8GB LPDDR4X** | **128GB UFS** | **Wi-Fi 5**

<a href="https://www.quectel.com/product/pi-h1-sbc-sg565d/" target="_blank" rel="noopener noreferrer">View on Quectel.com</a>

## Overview

The Quectel PI-SG565D (Pi H1) is a compact single-board computer built on the Qualcomm QCS6490 6nm SoC, delivering 12 TOPS of AI inference. It features dual MIPI CSI camera interfaces, dual display output (HDMI + USB-C DP), Wi-Fi 5, Bluetooth 5.0, and a Raspberry Pi-compatible ecosystem approach.
