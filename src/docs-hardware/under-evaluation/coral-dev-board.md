# Synaptics Coral Dev Board

Compact edge AI development board built by Grinn on the Synaptics Astra SL2619

**Synaptics Astra SL2619** | **1 TOPS NPU** | **Dual Cortex-A55** | **2GB DDR4** | **microSD**

<a href="https://grinn-global.com/blog/grinn-partners-synaptics-google-research-rs-synaptics-coral-dev-board" target="_blank" rel="noopener noreferrer">View on Grinn</a>

## Overview

The Synaptics Coral Dev Board is a compact edge AI development platform built by Grinn on its AstraSOM-261x system-on-module, powered by the Synaptics Astra SL2619 SoC with a 1 TOPS NPU. It targets low-power, multimodal edge AI — wearables, hearables, robotics, and smart home hubs — and runs the Synaptics Torq Edge AI software stack with support for Google's Coral NPU and Gemma open-source models. All user connectors sit on a single side for easy enclosure integration, with dedicated shunt resistors for power measurement of edge AI workloads.

- **Wearables and hearables** running low-power, always-on inference
- **Robotics and smart home hubs** with multimodal edge AI
- **Transformer-capable edge AI** prototyping with Gemma models on the Coral NPU
