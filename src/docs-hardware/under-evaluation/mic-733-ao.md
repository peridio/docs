Compact fanless AI computing system based on NVIDIA Jetson AGX Orin with up to 275 TOPS

**Up to 12-core Cortex-A78AE** | **Up to 275 TOPS** | **32/64GB LPDDR5** | **Fanless**

## Overview

The MIC-733-AO is a high-performance AI inference system built on the NVIDIA Jetson AGX Orin platform, delivering up to 275 TOPS in a compact fanless form factor. It offers rich I/O including 4x GbE with optional PoE, USB 3.2, and expansion via Mini-PCIe and M.2 slots. The system is Microsoft Azure Certified and supports Allxon 24/7 remote monitoring.
