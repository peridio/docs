# Thundercomm Rubik Pi 3

Affordable developer SBC with Qualcomm QCS6490 delivering 12 TOPS for on-device AI

**Octa-core Kryo 670** | **12 TOPS** | **8GB LPDDR4x** | **128GB UFS 2.2** | **100x75mm**

<a href="https://www.thundercomm.com/product/rubik-pi/" target="_blank" rel="noopener noreferrer">View on Thundercomm.com</a>

## Overview

The Thundercomm Rubik Pi 3 is a compact single-board computer built on the Qualcomm QCS6490 6nm SoC. It features a Raspberry Pi-like form factor with a 40-pin header, dual camera connectors, HDMI + USB-C display output, and Wi-Fi 5/BT 5.2. At approximately 4-8W power consumption, it delivers strong AI inference capability per watt.
