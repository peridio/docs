# SolidRun IIoT-200 RZ/G2L Gateway

Compact industrial IoT gateway with Renesas RZ/G2L, multi-protocol support, and DIN rail mounting

**Dual Cortex-A55 @ 1.2GHz** | **Mali-G31 GPU** | **Up to 2GB DDR4** | **128GB eMMC** | **120x80x30mm**

## Overview

The SolidRun IIoT-200 is a compact, power-efficient industrial IoT gateway built on the Renesas RZ/G2L SoM. It integrates a broad set of industrial protocols including RS232, RS485, CAN-FD, and dual Gigabit Ethernet in a tiny 120x80x30mm enclosure with DIN rail mounting. With onboard dual-band Wi-Fi 802.11ac and Bluetooth 5.0, it bridges operational technology (OT) with cloud connectivity.
