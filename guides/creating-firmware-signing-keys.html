Creating Firmware Signing Keys

This guide describes how to create firmware signing keys.

To learn more about Peridio firmware signing keys in general, see the firmware signing keys reference.

Prerequisites​

  • fwup CLI.
    • Last tested with version 1.9.1.
  • Peridio CLI.
    • Last tested with version 0.3.0.

Creation​

fwup -g

Submission to Peridio​

Web Console​

See the Peridio Web Console.

CLI​

peridio signing-keys create \
  --key value \
  --name value

The submitted key may now be used to sign firmware.

API​

Use the Peridio Admin API create key endpoint.

The submitted key may now be used to sign firmware.
