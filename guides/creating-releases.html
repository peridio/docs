Creating releases

This guide describes how to create releases.

To learn more about Peridio releases in general, see the releases reference.

Prerequisites​

  • Peridio CLI.
    • Last tested with version 0.4.0.

Create Release​

CLI​

peridio releases create \
  --bundle-id uuid \
  --cohort-id uuid

API​

Use the Peridio Admin API Create a release endpoint.

Web Console​

See the Peridio Web Console.
