Peridio Admin API (1.0.0)

Download OpenAPI specification:Download

The Peridio Admin API facilitates management of Peridio v1 (Cremini) resources.

Authentication

api_key

The header value must be Token ACCESS_TOKEN_VALUE.

Security Scheme Type API Key
Header parameter name: Authorization
