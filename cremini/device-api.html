Peridio Device API (1.0.0)

Download OpenAPI specification:Download

The Peridio Device API facilitates device-based operations within Peridio v1 (Cremini).

Authentication

Mutual TLS

The provided client certificate must be signed by a CA certificate registered with the Peridio Admin API.

Security Scheme Type mutualTLS

Devices

get device me

Authorizations:
Mutual TLS

    Responses
