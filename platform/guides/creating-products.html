Creating Products

This guide describes how to create produts.

To learn more about Peridio products in general, see the products reference.

Prerequisites​

  • Peridio CLI >= 0.11.0.

Create Product​

Web Console​

See the Peridio Web Console.

CLI​

peridio products-v2 create \
  --name name \
  --organization-prn prn

API​

Use the Peridio Admin API create-a-product-v2 endpoint.
